# Photos

Add your photos to this folder.

## Required files

| Filename       | Used for                                      |
|----------------|-----------------------------------------------|
| `cover.jpg`    | The hero photo — shown on the cover reveal    |
| `memory-01.jpg`| Memory 1 (polaroid + mosaic)                  |
| `memory-02.jpg`| Memory 2                                      |
| `memory-03.jpg`| Memory 3                                      |
| `memory-04.jpg`| Memory 4                                      |
| `memory-05.jpg`| Memory 5                                      |
| `memory-06.jpg`| Memory 6                                      |

## Tips

- **Cover photo** — use your best, most beautiful photo together.
  Landscape or portrait both work; it is displayed full-screen with a subtle warm filter.
- **Memory photos** — square or near-square crops look best inside the polaroid frame.
- **File format** — `.jpg` or `.png` both work. Keep files under 2MB each for fast loading.
- **More photos** — if you add more entries in `data/memories.js`, just add the matching
  file here with the same filename you used there.

## Optional: background music

Place a file called `ambient.mp3` here (in the `/photos/` folder root, alongside this README)
and the music button will play it when she taps it. Any calm, ambient, or piano track works.
Keep it under 5MB.
